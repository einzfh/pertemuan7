### 📱 Deskripsi Aplikasi
MyTaskList adalah aplikasi manajemen tugas harian yang dirancang dengan antarmuka *Dark Mode* yang modern dan intuitif. Aplikasi ini memungkinkan pengguna untuk mencatat rencana kegiatan, menandai tugas yang sudah selesai, serta memantau progres harian melalui fitur counter yang dinamis.

---

### ✅ Fitur & Requirement (Checklist)
Aplikasi ini telah mengintegrasikan seluruh konsep dasar dari P01 hingga P06:
- [x] **P01-P03:** Setup HP fisik & Layouting dengan Flexbox (View, Text, TouchableOpacity).
- [x] **P04 State Management:** Menggunakan `useState` untuk input teks dan daftar array task.
- [x] **P04 Conditional Rendering:** Menampilkan status "Task Selesai" dan Empty State jika list kosong.
- [x] **P05 Form Input & Validasi:** Input tugas dengan `TextInput`, `KeyboardAvoidingView`, dan alert validasi jika input kosong.
- [x] **P06 List Dinamis:** Menggunakan `FlatList` dengan `keyExtractor` dan `ListEmptyComponent`.
- [x] **Fitur CRUD:** Berhasil mengimplementasikan fungsi *Add* dan *Delete* task.
- [x] **Fitur Bonus:** Mark as Done (checklist) dan Counter progres tugas (X dari Y selesai).

### 🚀 Cara Menjalankan Project
1. Pastikan sudah menginstall Node.js dan Expo Go di HP fisik.
2. Clone atau buka folder project di VS Code.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Jalankan aplikasi:
   ```bash
   npx expo start -c
   ```
5. Scan QR Code yang muncul di terminal menggunakan aplikasi **Expo Go** (Android) atau **Kamera** (iOS).
